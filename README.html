Waycast Web Imager

This project implements a Web imager for waycasts and way models, a tool that transforms Waybreccian markup to HTML, so producing Web images of it. Effectively it extends the basic Breccia Web imager to accomodate waysource.

  • Definitions of ‘Web image’, ‘Waybreccian’ and ‘waysource’.
  • Basic Breccia Web imager.
  • Usage of the `waycast-web-image` command
